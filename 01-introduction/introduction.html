Introduction à ARIA - Exemples

1. Rôles ARIA

Ceci est une alerte importante!

Le rôle "alert" indique que ce contenu doit être annoncé immédiatement par les technologies d'assistance.

2. États ARIA

L'attribut "aria-pressed" indique l'état actuel du bouton (pressé ou non).

3. Propriétés ARIA

L'attribut "aria-label" fournit un label lisible pour les technologies d'assistance quand le texte visuel n'est pas suffisant.

4. ARIA avec contenu dynamique

L'attribut "aria-live" indique que les changements dans cette région doivent être annoncés par les technologies d'assistance.
